FFT & FIR Filter Design Case Studies

1. Fast Fourier Transform (FFT) Application

Scenario: You are given a noisy ECG signal and need to analyze its frequency content.

ECG Signal Model:

The ECG waveform can be approximated using a sum of sinusoids:
ECG(t) = A1*sin(2πf1t) + A2*sin(2πf2t) + ... + Noise

Noise Range: Powerline interference (50/60 Hz), baseline wander (0.1-0.5 Hz), muscle noise (20-300 Hz).

8-Point Radix-2 FFT Sample Frequencies: 0 Hz, 5 Hz, 10 Hz, 20 Hz, 40 Hz.

Bandpass Filter Range: 0.5 Hz to 100 Hz.

Hint: Apply a bandpass filter before FFT.

2. FIR Filter Design using Window Method

Scenario: Design a low-pass FIR filter for speech processing.

Specifications:

  • Cutoff Frequency: 4 kHz
  • Sampling Frequency: 16 kHz
  • Filter Order: 50
  • Window: Hamming

Hint: Compare Hamming and Blackman windows.

3. FFT-Based Convolution for Real-Time Audio Filtering

Scenario: Implement fast convolution for real-time audio filtering.

Specifications:

  • Sampling Rate: 44.1 kHz
  • Frame Size: 1024 samples
  • Low-pass FIR Filter: 51 coefficients

Hint: FFT reduces complexity from O(N²) to O(N log N).

4. Resampling and Frequency Response of Digital Filters

Scenario: Downsample a signal and analyze its frequency response.

Given Data:

  • Original Sampling Rate: 48 kHz
  • Target Sampling Rate: 16 kHz
  • Downsampling Factor: 3
  • Signal Frequency Components: 500 Hz, 3 kHz, 8 kHz, 12 kHz
  • Anti-Aliasing Filter: Low-pass FIR filter, Cutoff 7 kHz
  • Filter Order: 60 (Hamming window)

Hint: Use scipy.signal.resample or MATLAB’s resample() function.
